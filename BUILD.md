# Build

## Pre-requisite

* Java 11

## Running Unit Test

Run the following command to where you've cloned this repository:

```
gradlew clean check
```

## Build

Run the following command to where you've cloned this repository:

```
gradlew jar
```

> The **generated jar** file will be in the **libs directory** inside the **build directory** from where you've cloned the this repository.
